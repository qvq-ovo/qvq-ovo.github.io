1209. 带分数

说真的,如果不是放在DFS下面,我是想不到用DFS的.
不过话说过来,即使放在DFS下面,我也不知道怎么用BFS.555
只能说,BFS真神奇.

思路:
看到不要慌,整理下形式: n == a + b / c
a b c 三位正好将1 ~ 9 全用完.
所以呢, 我们只需要 弄出两个数来,剩下一个数用来判断是否成立.
因此,我们对形式进行变形: n * c - a * c == b.
所以我们使用b进行判断.
至于a c 我们使用DFS枚举所有情况.

y总建议: 在比赛中,如果没有想到这样,那就全排列,走最容易想到的写法.

总结: dfs,一个很神奇的算法.我只能说神奇.
